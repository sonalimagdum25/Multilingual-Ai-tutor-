# 🌍 Multilingual AI Tutor for Rural Education

AI-based web app that helps students learn in Marathi, Hindi, and English.

## Features
- Multilingual support
- Simple UI
- AI learning assistant
- Fast Node.js backend

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express

## How to Run
npm install  
npm start  

Then open frontend/index.html

- Frontend Developer

## Goal
Make education simple for rural students using AI.
